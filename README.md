 Library Management Dashboard

Overview

This project is a Library Management Dashboard built using **PostgreSQL** and **Microsoft Power BI. It provides interactive visualizations and insights into library operations, including books, members, and borrowing activities.

 Features

*  Total Books KPI
*  Total Members KPI
*  Total Borrow Records KPI
*  Books by Genre Analysis
*  Genre Distribution Dashboard
*  Most Borrowed Books
*  Borrow Count by Author
*  Monthly Borrow Trends
*  Detailed Books and Member Tables

 Technologies Used

* PostgreSQL
* SQL
* Microsoft Power BI

 Database Tables

* **Books**
* **Members**
* **Borrow Records**

 Dashboard Pages

1. Overview Dashboard
2. Books Analysis
3. Members Analysis
4. Borrow Activity

 Key Insights

* Analyze the distribution of books by genre.
* Identify the most borrowed books and authors.
* Monitor borrowing activity over time.
* Explore member and transaction details through interactive reports.
